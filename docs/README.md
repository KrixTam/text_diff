# 文本对比大师（通用 AI 版本）

专业级文本文件对比工具，为文档编辑、开发者及内容创作者提供清晰、智能的文本版本比对体验。当前版本已改造成通用 AI SDK 调用，支持通过环境变量灵活配置 API Key、Base URL 与 Model。

## 核心功能

- 全格式支持：支持 Markdown (.md)、代码文件 (.js, .ts, .py, .java 等) 以及任意纯文本 (.txt)
- 双模式比对：支持分栏视图 (Split) 与混合视图 (Unified) 两种展示方式
- 差异高亮：支持行级与单词级的精准差异识别，通过深浅色背景显著标出变更内容
- AI 智能总结：基于通用 OpenAI SDK（或兼容 OpenAI 的网关），自动分析并总结两个版本之间的核心变更（需代码开启相关功能）
- 一键导出：支持将对比结果以高清晰度 PNG 图片形式下载
- 隐私与安全：纯前端文件处理，文件内容仅在本地进行差异计算

## 快速上手

1. 上传文件：分别在“原始文件”和“新文件”区域点击上传需要比对的文件
2. 开始对比：点击“开始对比”按钮，系统立即渲染差异
3. 切换视图：根据需要使用顶部工具栏切换不同的对比模式
4. 保存结果：点击“下载图片”保存当前可视化的对比全貌

## 环境配置

在 `.env` 或 `.env.local` 中设置以下变量以启用 AI 总结：
- `OPENAI_API_KEY`：模型服务的 API Key
- `OPENAI_BASE_URL`：可选；使用兼容 OpenAI 的服务网关时配置
- `OPENAI_MODEL`：模型名称（如 `gpt-4o-mini`、`gpt-4o` 或其他兼容型号）

## 技术栈

- 框架：React 19
- 样式：Tailwind CSS
- 图标：Lucide React
- 比对算法：jsdiff
- AI 引擎：OpenAI SDK（或兼容 OpenAI 的网关）
- 渲染导出：html-to-image
